## 👋 About Me

I'm a CS graduate with a genuine curiosity about how software systems are built and delivered at scale. My interest started with hardware and performance, and grew into a focus on cloud infrastructure — how applications move from local code to globally distributed systems, how services communicate reliably, and how thoughtful architecture makes them faster and more resilient.

I'm especially drawn to **cloud engineering** and **backend systems**: storage, networking, security, and the infrastructure layer that makes modern applications work. My goal is to grow into a cloud-focused engineering role where I can build dependable systems and keep learning in a hands-on environment.
